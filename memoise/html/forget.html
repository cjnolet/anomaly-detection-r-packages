forget {memoise}R Documentation

Forget past results. Resets the cache of a memoised function.

Description

Forget past results. Resets the cache of a memoised function.

Usage

forget(f)

Arguments

f

memoised function

See Also

memoise, is.memoised

Examples

memX <- memoise(function() { Sys.sleep(1); runif(1) })
# The forget() function
system.time(print(memX()))
system.time(print(memX()))
forget(memX)
system.time(print(memX()))
