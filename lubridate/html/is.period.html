Is x a period object?

Description

Is x a period object?

Usage

  is.period(x)

Arguments

x

an R object

Value

TRUE if x is a period object, FALSE otherwise.

See Also

is.instant, is.timespan, is.interval, is.duration, period

Examples

is.period(as.Date("2009-08-03")) # FALSE
is.period(new_period(months= 1, days = 15)) # TRUE
