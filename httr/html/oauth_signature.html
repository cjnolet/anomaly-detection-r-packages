oauth_signature {httr}R Documentation

Generate oauth signature.

Description

For advanced use only. Occassionally needed for sites that use some components of the OAuth spec, but not all of them (e.g. 2-legged oauth)

Usage

oauth_signature(url, method = "GET", app, token = NULL,
  token_secret = NULL, ...)

oauth_header(info)

Arguments

url,method

Url and http method of request.

app

oauth_app object representing application.

token,token_secret

OAuth token and secret.

...

Named argument providing additional oauth parameters (e.g. oauth_callback or oauth_body_hash). Names should not include the "oauth_" prefix - this will be automatically included.

Value

A list of oauth parameters.
