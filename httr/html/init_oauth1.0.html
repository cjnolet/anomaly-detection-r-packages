init_oauth1.0 {httr}R Documentation

Retrieve OAuth 1.0 access token.

Description

See demos for use.

Usage

init_oauth1.0(endpoint, app, permission = NULL,
  is_interactive = interactive())

Arguments

endpoint

An OAuth endpoint, created by oauth_endpoint

app

An OAuth consumer application, created by oauth_app

permission

optional, a string of permissions to ask for.

is_interactive

Is the current environment interactive?
