release_checks {devtools}R Documentation

Custom devtools release checks.

Description

This function performs additional checks prior to release. It is called automatically by release().

Usage

release_checks(pkg = ".", built_path = NULL)

Arguments

pkg

package description, can be path or package name. See as.package for more information. If the DESCRIPTION file does not exist, it is created using create_description.
