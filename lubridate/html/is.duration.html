Is x a duration object?

Description

Is x a duration object?

Usage

  is.duration(x)

Arguments

x

an R object

Value

TRUE if x is a duration object, FALSE otherwise.

See Also

is.instant, is.timespan, is.interval, is.period, duration

Examples

is.duration(as.Date("2009-08-03")) # FALSE
is.duration(new_duration(days = 12.4)) # TRUE
