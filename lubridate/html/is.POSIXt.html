Is x a POSIXct or POSIXlt object?

Description

Is x a POSIXct or POSIXlt object?

Usage

  is.POSIXt(x)

Arguments

x

an R object

Value

TRUE if x is a POSIXct or POSIXlt object, FALSE otherwise.

See Also

is.instant, is.timespan, is.Date

Examples

is.POSIXt(as.Date("2009-08-03")) # FALSE
is.POSIXt(as.POSIXct("2009-08-03")) # TRUE
